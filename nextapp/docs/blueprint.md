# **App Name**: ShadowTrace

## Core Features:

- Secure Authentication and Wallet Connection: Implement secure user authentication with email/password and MetaMask wallet connection.
- Investigation Workspace: Enable a focused investigation for a single wallet/transaction with graph visualizations and OSINT data.
- Multi-Chain Graph Visualizations: Provide high-density analytics across multiple graphs and chains with visual comparison and global intelligence overlay.
- OSINT Fusion and Correlation: Deep OSINT correlation and entity management capabilities.
- AI Risk Assessment: AI-driven risk assessment of wallets, flagging potentially suspicious activities.
- Report Viewing: Implement in-app report viewer mirroring the Crimson Syndicate PDF, including Executive Summary, Subject Profile, On-Chain Analysis, OSINT Fusion, and more.
- Global Threat Intelligence Overlay: Visualize threat actors and their activities on a global map.

## Style Guidelines:

- Primary color: Neon blue (#13D2FF), creating a high-tech, intelligence-focused aesthetic.
- Background color: Deep navy (#191D2B), establishing a dark, cyber-intelligence theme.
- Accent color: Amber (#FFB347) to indicate warnings, critical alerts shown in Red (#FF4545) and safe indicators in Green (#32CD32).
- UI font: 'Inter' (sans-serif) for a clean and readable interface.
- Data font: 'Roboto Mono' (monospace) for code blocks and graph data.
- Use minimalist, serious icons representing security, data, and network concepts.
- Implement a 12-column grid layout with a persistent left sidebar, main content panel, and a dynamic right details drawer.
- Employ subtle animations, such as fade/slide on drawers and a rotating chain ring loader.
